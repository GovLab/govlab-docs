Network of Innovators

History

Ruby on Rails version of Network of Innovators with updated interface and styleguide. Data Justice Network is a clone of NOI.

Prerequisites

  • Install Rails
  • Set up Bundler to manage gems
  • Set up PostgreSQL

Getting Started

  • Clone the repo: NOI3

Installing

  • Fetch and update bundled gems by running bundle install

Development

  • Run the server with rails s. Your server will run at http://localhost:4000/
  • Pull environment variables from Heroku.

Discourse

NOI’s instance of Discourse runs on a Digital Ocean droplet. Add your ssh key and then.

ssh root@discuss.networkofinnovators.org to access the production instance

or

ssh root@discourse-dev.networkofinnovators.org to access the development instance

Features

Discourse

By default, the discussions in Discourse are threaded. We customized the feed on the index to show us single replies.

Username Validation

Username rules are synced up to the rules set in Discourse. CODE

User Skills

User skills are collected through a checklist that is entered into the DB as a Teachable or Learnable skill. Categories, skill areas, and skills were uploaded from a CSV.

User avatars

User avatars are uploaded with Paperclip and stored with AWS.

Matches

Matches are determined using user’s Teachables and Learnables. CODE

Troubleshooting Discourse

  • If Discourse is down, you can ssh into the machine and enter reboot to do a complete reboot.

  • If you want to install your own instance of discourse, follow the instructions here.

  • SSO is enabled so any changes to email and username validation done in NOI needs to be mirrored in Discourse. Most errors occur when usernames in the Postgres DB and Discourse do not match. If a username has been previously taken or fails validation Discourse but works in NOI, the Discourse username will be adjusted.
